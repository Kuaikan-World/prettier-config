# prettier-config

快看前端团队 Prettier 统一配置项

## 安装

### NPM

```bash
>$ npm install @kuaikan/prettier-config --save-dev
```

### Yarn

```bash
>$ yarn add @kuaikan/prettier-config -D
```

### Pnpm

```bash
>$ pnpm add @kuaikan/prettier-config -D
```

## 使用方式

```bash
>$ echo '"@kuaikan/prettier-config"' > .prettierrc
```
